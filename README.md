## 🚀 Getting Started

Want to run any project locally? Here's how:

**Prerequisites:**
- [Node.js](https://nodejs.org/) (v18 or above recommended)
- npm (comes with Node)

**Steps:**

```bash
# 1. Clone the repo
git clone https://github.com/your-username/your-repo-name.git

# 2. Go into the project folder you want to run
cd 03-projects/project-name

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

> Note: Some folders are just `.js` / `.html` files — no installation needed for those, just open them directly.
